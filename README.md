     CinemaniaApp
</h1>

## Description
<p> Simple movie search app using the Omdb Api.</p>


## Technologies and Libraries
- Protocol Oriented Programming
- URLSession
- Programmatic UI
- MVVM
- Kingfisher
- AutoLayout
- Firebase Analytics
- Firebase Remote Config
